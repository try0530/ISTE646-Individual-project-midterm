About This Website

This website include the information about taipei city, food, activities, festival, and more.
During this project, I try to structure the content so that can make the website looks consistency.
The main introduction pages are almost using the same CSS design. And the detail pages are using the same CSS design strucutre. Most of the content section is repeated. In my thought, I think it can change to php functions to let them repeat automatically.

The navbar, trying to build two navbar to suit different screen size. This took me lots of time to make them fit in different screen sizes.

During this project, I think I was too ambitious to build too many pages which made me run out of time to think properly about the design. I am planning to keep working on this project to make it better before the second individual project start.

There are some pages that I plan to build but didn't have time to build yet. I still create the link to there, will build them later.
